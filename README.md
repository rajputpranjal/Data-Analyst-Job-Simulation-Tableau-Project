# Data-Analyst-Job-Simulation-Tableau-Project

# Project Overview

This project is part of a Data Analyst Job Simulation where I analyzed real-world industrial telemetry data using Tableau.
The objective was to identify machine breakdown patterns, uncover inefficiencies, and provide actionable insights to improve factory operations.

# Factories Analyzed
Daikibo Factory Meiyo (Tokyo)

Daikibo Factory Seiko (Osaka)

Daikibo Berlin

Daikibo Shenzhen

# Dashboard Features
Interactive Tableau Dashboard

Failure trend analysis over time

Factory-wise breakdown comparison

Machine-level failure insights

Global performance visualization

# Key Insights
 Seiko (Osaka) recorded the highest number of machine breakdowns
 LaserWelder machines were the most failure-prone
 Identified clear patterns in failure trends and peak downtime periods
